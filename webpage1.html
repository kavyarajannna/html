Supplier Monitoring Process
Process ID ACQ.4
Process name Supplier Monitoring
Process purpose The purpose of the Supplier Monitoring Process is to track and assess the performance of the supplier against agreed requirements.
Process outcomes As a result of successful implementation of this process:
1) Joint activities are performed as needed;
2) All agreed information is communicated regularly;
3) Supplier performance is monitored against agreements;
4) Changes to agreements are negotiated and documented.
Base Practices ACQ.4.BP1: Agree on and maintain joint processes, interfaces, and information to be exchanged. Establish and maintain agreements on responsibilities, type and frequency of activities, communications, etc. [OUTCOME 1, 2, 4]

NOTE 1: Joint processes and interfaces include project management, requirements management, etc.
NOTE 2: Joint activities should be mutually agreed.
NOTE 3: Customer refers to the assessed party, supplier refers to the supplier of the assessed party.

ACQ.4.BP2: Exchange all agreed information using defined interfaces. [OUTCOME 1, 2, 3]
NOTE 4: Agreed information includes all relevant work products.

ACQ.4.BP3: Review technical development with the supplier regularly, covering technical aspects, problems, risks, and open items. [OUTCOME 1, 3, 4]

ACQ.4.BP4: Review supplier progress regarding schedule, quality, and cost regularly. Track open items and perform risk mitigation. [OUTCOME 1, 3, 4]

ACQ.4.BP5: Act to correct deviations from agreed objectives. Negotiate changes and document them in agreements. [OUTCOME 4]
